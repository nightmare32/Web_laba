Опис предметної галузі

⇦ Повернутися на головну сторінку

Назва теми: веб-застосунок для перегляду розкладу та результатів футбольних змагань

Предметом дослідження є: огляди існуючих веб-застосунків для перегляду результатів матчів

Постановка задачі є: розробка веб додатку у якому будуть зібрані результати найбільш популярних футбольних ліг та турнірів. Також користувачі зможуть ділитися своїми враженями від матчів.

Метою цієї роботи є: розробка простого та приємного на вид веб-застосунку для звичайних вболівальників футболу. Цей веб-застосунок повинен полешить та зробить зручним процес пошуку результату матчу із найцікавіших турнірів.

Опис бізнес-логіки предметної галузі:

  • Адміністратор:
    • Додавати футбольні події;
    • Видаляти футбольні події;
    • Редагувати футбольні події;
    • Вводити результати матчів;
  • Користувач:
    • Реєстрація;
    • Авторизація;
    • Переглядати розклади змагань;
    • Шукати команду по назві;
  • Зарегестрований користувач:
    • Додавати будь-яку команду у улюблені;
    • Залишати коментарі до матчу;

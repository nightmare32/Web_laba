Ідентифікатор

Ідентифікатор (названий також "ID селектор") визначає унікальне ім'я елемента, яке використовується для зміни його стилю та звернення до нього через скрипти.

Приклад використання: 

        <style>
        #popup {
            background-color: red;
            padding: 10px;
            border: 1px solid black;
            width: 50%;
            margin: auto;
        }
    </style>

 <div id="popup" class="removeOn">
        <div class="popup_body">
            <form>
                <h2>Реєстраційна форма</h2>

                <label for="FName">Ім'я:</label><br>
                <input class="FName" type="text" name="FName" id="FName"><br>
                <label for="LName">Прізвище:</label><br>
                <input class="LName" type="text" name="LName" id="LName"><br>
                <label for="Password">Пароль:</label><br>
                <input class="PWord" type="password" name="Password" id="Password"><br><br>

                <button type="submit" id="submit">Submit</button>
                <button type="reset">Reset</button>
                <button id="close">Close</button>
            </form>
        </div>
    </div>

Селектор ідентифікатора

Ідентифікатор – особливий вид селектора, яким позначається один елемент на сторінці. Щоб вказати селектор ідентифікатор, у CSS необхідно використовувати конструкцію #ім'я_ідентифікатора.

Приклад 

        <p id="example"> Lorem, ipsum dolor sit.</p>
       
            #example{
                color:darkgoldenrod;
                font-size: 20px;
            }

Lorem ipsum dolor sit.

Дочірні та контекстні селектори

Дочірнім називається елемент, який безпосередньо розташовується всередині батьківського елементу. 


        <div>
            <h2>Lorem ipsum dolor sit amet.</h2>
        </div>
        
        div h2{
            color: red;
            font-size: 10px;
        }
